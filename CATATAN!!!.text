🧱 Migrasi SQLite
Laravel 11 sudah siap digunakan dengan SQLite, tapi pastikan langkah berikut dilakukan:
1. 	Lakukan bash command berbeda terminal
Contoh:
-- Terminal BASH
php artisan serve

-- Terminal PowerShell
npm run dev

2. 	Lakukan migrasi:
php artisan migrate

3. 	Aktifkan ekstensi SQLite di PHP:
composer update

3. 	Buka file php.ini dan pastikan:
extension=pdo_sqlite
extension=sqlite3

📎 Untuk pengguna Laragon atau XAMPP, pastikan konfigurasi ini diaktifkan agar migrasi bisa berjalan lancar.
